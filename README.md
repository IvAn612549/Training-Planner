# Training Planner

**Автор:** Иван Хребто (замените на своё имя)

## Описание

Training Planner — это простое десктопное приложение для планирования тренировок.  
Оно позволяет:

- Добавлять тренировки с указанием даты, типа и длительности.
- Просматривать список всех тренировок.
- Фильтровать тренировки по типу или дате.
- Автоматически сохранять и загружать данные в файл `trainings.json`.

## Примеры использования

### Запуск приложения

```bash
python training_planner.py
